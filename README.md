# 📈 Stock Trend Prediction

A machine learning project that analyzes historical stock market data and predicts future price trends using predictive models and data visualization techniques.

---

## 📌 Overview

The project demonstrates financial data preprocessing, trend analysis, visualization, and machine learning-based forecasting for stock prices.

---

## ✨ Features

- Historical stock analysis
- Data preprocessing
- Trend visualization
- Price prediction
- Performance evaluation

---

## 🛠 Tech Stack

- Python
- TensorFlow
- Pandas
- NumPy
- Matplotlib

---
